Ex01 : Django, framework web.

Qu'est-ce que Django ?

Django est un framework web open source écrit en Python. Il suit le modèle architectural MTV (Model-Template-View), une variante du modèle MVC (Model-View-Controller). Django est conçu pour permettre le développement rapide d'applications web sécurisées et maintenables.

Historique

Django a été créé à l'automne 2003, lorsque les programmeurs web du journal Lawrence Journal-World, Adrian Holovaty et Simon Willison, ont commencé à utiliser Python pour construire des applications web.

Le framework a été nommé d'après le guitariste de jazz Django Reinhardt. La première version publique (0.91) a été publiée en juillet 2005, et la version 1.0 a été publiée en septembre 2008.

Depuis, Django a connu de nombreuses versions majeures, ajoutant des fonctionnalités comme les migrations de base de données, un meilleur support des applications réutilisables, et des améliorations de performance.

Philosophie

Django suit plusieurs principes clés :

  • DRY (Don't Repeat Yourself) : Éviter la duplication de code.
  • Convention over Configuration : Des valeurs par défaut sensées.
  • Loose Coupling : Les différentes couches de l'application sont indépendantes.
